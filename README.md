==

Find your public or private IP address.

## :minidisc: Install [![AUR package](https://img.shields.io/aur/version/myip)](https://aur.archlinux.org/packages/myip)

```sh
make DESTDIR=/ PREFIX=/usr install
```

## "woman_astronaut: Usage

```sh
myip <scope>
```

### public

```sh
myip public
```

### private

```sh
myip private
```

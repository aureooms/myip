# myip

> Print your public or private IP address.

[![AUR package](https://img.shields.io/aur/version/myip)](https://aur.archlinux.org/packages/myip)

## Install

```sh
make DESTDIR=/ PREFIX=/usr install
```


## Usage

```sh
myip <scope>
```

### public

```sh
myip public
```

### private

```sh
myip private
```
